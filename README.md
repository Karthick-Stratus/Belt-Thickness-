# Belt Thickness Analytics Dashboard

An industrial-grade, interactive web dashboard for visualizing and analyzing manufacturing belt thickness data.

## 🚀 Live Demo
You can view the live dashboard here: [https://karthick-stratus.github.io/Belt-Thickness-/](https://karthick-stratus.github.io/Belt-Thickness-/)

## ✨ Key Features
- **Real-time Interaction:** Zoom, pan, and hover over data points for precise measurements.
- **Dynamic Trend Analysis:** Adjustable Moving Average (MA) window to smooth data and identify long-term trends.
- **Envelope Mapping:** Automatic calculation of upper and lower boundary curves (peaks and valleys).
- **Frequency Distribution:** Integrated bar chart to analyze value concentration and spot common outliers.
- **Report Generation:** Export the entire dashboard as a professional PDF report.
- **Responsive Design:** Optimized for full-screen desktop viewing and mobile devices.

## 🛠️ Technology Stack
- **HTML5/JS:** Core structure and logic.
- **Tailwind CSS:** Modern, utility-first styling for a premium "glassmorphism" UI.
- **Plotly.js:** High-performance data visualization library.
- **html2pdf.js:** Client-side PDF generation.
- **Lucide Icons:** Clean, consistent industrial iconography.

## 📊 Data Specifications
The dashboard currently visualizes:
- **X-Axis:** Distance from Edge 1 (mm)
- **Y-Axis:** Belt Thickness Reading (mm)

## 📖 How to Use
1. **Control Panel:** Use the sidebar to toggle data layers (Raw Data, MA, Envelopes).
2. **Trend Window:** Drag the slider to adjust the Moving Average smoothing factor.
3. **Zooming:** Use the manual +/- buttons or scroll/box-select on the charts directly.
4. **Export:** Click "Download PDF Report" to save a snapshot of the current analysis.
